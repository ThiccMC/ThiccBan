# ThiccBan-Next
ThiccMC Ban List. Using Next.js based on default Litebans' page.

**WARNING**: This page is customized only for ThiccMC Network. You are free to use it but use at your own risk!

### Development
```
npm i --s
npm run dev
```

### Build
```
npm run build
npm start
```

### Requirements
- Pre-setup MySQL Litebans Database
